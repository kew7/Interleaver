================================================================================
    MICROSOFT FOUNDATION CLASS LIBRARY : Interleaver_Dialog Project Overview
===============================================================================

Простая программа, реализующая побитовое перемежение входного файла (траспонирование матрицы) и запись результата в файл.
Диапазон допустимых значений количества строк и столбцов при транспонировании матрицы - от 2 до 256.
