================================================================================
    MICROSOFT FOUNDATION CLASS LIBRARY : Interleaver_Dialog Project Overview
===============================================================================

A simple program that implements bitwise interleaving of the input file (matrix transposition) and writes the result to a file.
The range of values for the number of rows and columns when transposing a matrix is from 2 to 256.

Простая программа, реализующая побитовое перемежение входного файла (траспонирование матрицы) и запись результата в файл.
Диапазон допустимых значений количества строк и столбцов при транспонировании матрицы - от 2 до 256 (при желании можно изменить).
